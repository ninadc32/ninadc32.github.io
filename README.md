## Instructions

### Setup

```shell
git clone https://github.com/ninadc32/ninadc32.github.io.git
cd ninadc32.github.io.git
```

If you use [nvm](https://github.com/nvm-sh/nvm) or [fnm](https://github.com/Schniz/fnm) (optional) -

```shell
nvm install
nvm use
```

Or -

```shell
fnm install
fnm use
```

Install and run -

```shell
yarn
yarn start
```
